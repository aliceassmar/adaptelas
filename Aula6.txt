    # = - Usado para declaração de variável
    # == - valor de um é igual ao do outro
    # != - Valor de um é diferente do outro
    # > - maior que
    # < - menor que
    # + - soma
    # - - subtração
    # === - igual ou do mesmo tipo 
    # >= ou <= - maior ou menor igual
    # % - resto da divisão. 
Retorna integer ou decimal (Na verdade só retorna 0 ou 1 - 0 se não tiver resto e 1 se tiver, mas não informa qual é a diferença)
    # / - divisão
    # ** - elevado a
    # && - 'e' as duas condições são verdadeiras
    # || - 'ou' ou uma ou outra condição é verdadeira. Pelo menos uma delas tem que ser verdadeira
