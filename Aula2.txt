# tags padrões
<!DOCTYPE html> # versão do html
<html></html> # demarca o documento, tudo que tiver dentro dela vai ser interpretado pelo navegador
<head></head> # Trazer a informação para a gente
<title></title> # título da guia, do navegador
<meta/> # metadados
<body><body/> # Tudo que for fazer no site
<html lang="en"> # pt-BR para especificar o pt do Brasil
